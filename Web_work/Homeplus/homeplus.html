Period

Review

  • 두개의 언어로 만든 Animation

    •  4개의 영역과, 16개의 원 그리고 ::after까지 많은 요소에 원하는 모양과, 움직임을
    구현하려하니 코드가 너무 복잡하고 길어져 원하는 이미지를 구현하기 어려웠습니다.     이를 해결하기위해 여러개의 CSS @keyframes와, class명을 만들어 중복 부여하였고
    CSS를 이용해 다양한 모양과 움직이는 애니메이션을 제작할 수 있었습니다.


    •  시간차 텍스트 애니메이션은 JavaScript언어를 이용하여 만들 수 있었습니다.
    forEach()문을 이용하여, 배열 인자들을 돌아가며 선택할 수 있도록 한 뒤
    setTimeout(),setInterval()을 이용하여 반복 시간값을 주어 구현하였습니다

  • API와의 해프닝과 배움

    오류는 없지만 지도가 뜨지 않았고, 이를 해결하기위해 다양한 구글링을 하여 찾은 결과는 허무했습니다
    서버에 올릴땐 url 스킴이 없어도 되지만, 로컬에서 확인할때는 https://가 필요했던 것 이었습니다.
    작은 해프닝 끝에 kakao API를 이용하여 지도를 불러오고, 좌표를 입력하여
    해당 좌표에 마커를 표시하는 등 API 사용법에 대해 공부할 수 있었습니다.
